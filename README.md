# Simulador Undo/Redo con Pilas en Java

## Objetivo del proyecto
Comprender qué es una pila (Stack) e implementarla manualmente en Java,
aplicándola en un simulador de Deshacer y Rehacer texto.

---

## ¿Qué es una pila?
Una pila es una estructura de datos que funciona como una torre de platos.

El último elemento que entra es el primero que sale.

Esto se llama LIFO (Last In, First Out).

Operaciones básicas:
- push(): agrega un elemento arriba
- pop(): elimina el elemento de arriba
- peek(): muestra el elemento superior
- isEmpty(): verifica si está vacía

---

## ¿Cómo se aplica a Undo/Redo?
- Cuando escribo texto → se guarda en la pila principal
- Cuando deshago → pasa a la pila secundaria
- Cuando rehago → vuelve a la pila principal

Así se simula cómo funcionan los editores de texto.
